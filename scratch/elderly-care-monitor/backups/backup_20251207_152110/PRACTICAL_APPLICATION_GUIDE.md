## 🚀 3가지 적용 방법

### 방법 1: 개인 가정 설치 (DIY)

**대상**: 독거노인 가구, 맞벌이 자녀 가구

**장점**:
- ✅ 저렴한 비용 ($180-250)
- ✅ 빠른 설치 (1-2시간)
- ✅ 완전한 통제권

**단계**:
1. 하드웨어 구매 → [체크리스트 참조](QUICK_START_CHECKLIST.md)
2. 침실 천장/벽면 설치
3. 보호자 앱 연동
4. 1주일 테스트 및 최적화

**예상 비용**:
- 초기: $180-250 (약 24-33만원)
- 월간: $17-33 (약 2-4만원)

---

### 방법 2: 구독 서비스 (렌탈)

**대상**: 기술에 익숙하지 않은 가족, 간편함 선호

**장점**:
- ✅ 초기 비용 없음
- ✅ 전문 설치 서비스
- ✅ 정기 점검 포함
- ✅ 하드웨어 교체 보증

**단계**:
1. 웹사이트에서 구독 신청
2. 전문가 방문 설치 (1시간)
3. 보호자 앱 교육
4. 24/7 모니터링 시작

**예상 비용**:
- 월 구독료: $29.99 (약 4만원)
- 설치비: 무료
- 유지보수: 포함

---

### 방법 3: 요양원/요양병원 도입 (B2B)

**대상**: 요양 시설, 병원, 지자체

**장점**:
- ✅ 대량 할인 (30-50%)
- ✅ 중앙 관리 시스템
- ✅ 직원 교육 포함

**단계**:
1. 파일럿 테스트 (10-20명)
2. 효과 검증 (3개월)
3. 전체 도입 결정
4. 직원 교육 및 운영

**예상 비용**:
- 대당: $150 (100대 이상)
- 관리 소프트웨어: $500/월
- 교육: 1회 $2,000

## 🚧 자주 발생하는 문제와 해결책

### 문제 1: 오탐이 너무 많아요

**원인**:
- 반려동물 움직임
- 선풍기/에어컨 바람
- 민감도 설정 과다

**해결책**:
```
1. 감지 민감도 낮추기 (7 → 5)
2. 반려동물 출입 제한 (침실 문 닫기)
3. 센서 각도 조정 (침대만 향하도록)
4. 캘리브레이션 재수행
```

### 문제 2: 낙상을 감지하지 못했어요

**원인**:
- 감지 범위 밖에서 낙상
- 민감도 설정 부족
- 센서 고장

**해결책**:
```
1. 감지 민감도 높이기 (7 → 9)
2. 센서 위치 재조정
3. 추가 센서 설치 (거실, 화장실)
4. 센서 연결 상태 확인
```

### 문제 3: 알림이 오지 않아요

**원인**:
- 인터넷 연결 끊김
- 앱 알림 권한 미설정
- 스팸 필터링

**해결책**:
```
1. WiFi 연결 확인
2. 앱 알림 권한 설정 (설정 > 알림)
3. 이메일 스팸 폴더 확인
4. 전화번호 정확성 확인
```

### 문제 4: 배터리가 자꾸 방전돼요

**원인**:
- 정전 또는 전원 불안정
- UPS 미설치

**해결책**:
```
1. 안정적인 전원 콘센트 사용
2. UPS(무정전 전원장치) 설치
3. 전원 케이블 점검
4. 백업 배터리 추가
```

## ✅ 실행 체크리스트

### 즉시 실행 가능 (오늘)

- [ ] 이 문서 읽기 완료
- [ ] 적용 방법 결정 (DIY / 구독 / B2B)
- [ ] 예산 확인
- [ ] 가족 회의 (동의 구하기)

### 1주일 내

- [ ] 하드웨어 구매 또는 구독 신청
- [ ] 설치 위치 결정
- [ ] 보호자 명단 작성
- [ ] 정부 지원 사업 확인

### 1개월 내

- [ ] 설치 완료
- [ ] 보호자 교육 완료
- [ ] 1주일 테스트 완료
- [ ] 최적화 완료

### 3개월 내

- [ ] 안정적 운영
- [ ] 데이터 분석 및 개선
- [ ] 추가 센서 고려
- [ ] 이웃에게 추천
