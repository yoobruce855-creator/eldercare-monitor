# 🏥 ElderCare Monitor - 비접촉 생체 신호 모니터링 시스템

## 📋 프로젝트 개요

**목표**: "비접촉 방식으로 3가지 핵심 상태(정상, 낙상, 무호흡)를 90% 정확도로 감지하고 즉시 보호자에게 알림을 보낸다."

mmWave 레이더 센서를 활용한 실시간 생체 신호 모니터링 및 이상 징후 감지 시스템 MVP

## ✨ 핵심 기능

### 1. 생체 신호 감지
- **비침습적 모니터링**: 사생활 침해 없이 호흡수, 심박수 감지
- **mmWave 센서**: Point Cloud 데이터 수집 및 노이즈 필터링
- **실시간 처리**: 초당 10회 데이터 샘플링

### 2. 이상 징후 패턴 인식
- **낙상 감지**: 심박수 급증 + 낮은 움직임 패턴
- **무호흡 감지**: 5분 이상 움직임 없음 + 낮은 호흡수
- **엣지 AI 모델**: 사전 학습된 패턴 인식 알고리즘

### 3. 긴급 알림 발송
- **10초 대기 타이머**: 오작동 방지 로직
- **즉시 알림**: 이상 징후 감지 시 10초 이내 알림 발송
- **다중 채널**: 푸시 알림, SMS, 이메일

## 🔄 작동 로직 플로우

```
1. 데이터 수집 (mmWave 센서)
   ↓
   - Raw Point Cloud 데이터 수집 (초당 10회)
   - 노이즈 필터링
   
2. 엣지 AI 분석
   ↓
   - 데이터 전처리 (호흡/심박 그래프 변환)
   - AI 패턴 분류: 정상 / 낙상 / 무호흡
   
3. 응급 상황 판단
   ↓
   - IF (낙상 또는 무호흡 감지)
   - THEN 10초 대기 타이머 작동
   - IF (10초 내 움직임 없음)
   - THEN 응급 상황 최종 판단
   
4. 클라우드 서버 연동
   ↓
   - 응급 데이터 전송
   - 보호자 목록 확인
   
5. 사용자 관제 (모바일 앱)
   ↓
   - 푸시 알림 발송
   - 위치 및 생체 신호 변화 추이 표시
```

## 🛠️ 기술 스택

### 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS3**: 다크 모드, 그라디언트, 애니메이션
- **Vanilla JavaScript**: 실시간 데이터 처리

### 시뮬레이션 모듈
- **SensorModule**: mmWave 센서 데이터 생성 및 필터링
- **EdgeAIModule**: 패턴 인식 및 분류
- **EmergencyProcessor**: 응급 상황 판단 및 알림

### 실제 하드웨어 (향후 통합)
- **센서**: mmWave 레이더 (60GHz)
- **엣지 디바이스**: Raspberry Pi 4 / Jetson Nano
- **클라우드**: AWS IoT Core / Google Cloud IoT

## 🧪 테스트 시나리오

시스템 시작 후 자동으로 다음 시나리오가 실행됩니다:

1. **5초 후**: 낙상 시뮬레이션
   - 알림 배너 표시
   - 10초 대기 타이머 시작
   
2. **15초 후**: 정상 상태 복귀
   - 알림 취소
   - 정상 모니터링 재개

### 수동 테스트 함수
브라우저 콘솔에서 실행:
```javascript
simulateFall()    // 낙상 시뮬레이션
simulateApnea()   // 무호흡 시뮬레이션
simulateNormal()  // 정상 상태 복귀
```
